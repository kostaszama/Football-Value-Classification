# Football Player Value Classification (FIFA 15 - EA Sports FC 24)

This project focuses on predicting and classifying the economic value of football players into three categories (**Low**, **Mid**, **High**) based on their in-game performance attributes and statistics.

## Project Overview
The analysis uses the "EA Sports FC 24 Complete Player Dataset" from Kaggle, covering player data from FIFA 15 up to EA Sports FC 24. The goal is to determine how well a player's physical and technical skills can reflect their market value class.

## Key Features
* **Comprehensive Preprocessing:** Cleaning of the multi-year dataset, handling missing values, and encoding categorical data such as player positions and work rates.
* **Exploratory Data Analysis (EDA):** Statistical analysis and visualizations of player wages and attribute distributions.
* **Machine Learning Implementation:** * **Decision Trees**
    * **Support Vector Machines (SVM)**
    * **Multi-layer Perceptron (MLP) Neural Networks**
* **Evaluation:** Comparative analysis using accuracy scores and normalized Confusion Matrices to evaluate how each model handles the three value classes.

## Dataset
The data used in this notebook is sourced from the [EA Sports FC 24 Complete Player Dataset](https://www.kaggle.com/datasets/stefanoleone992/ea-sports-fc-24-complete-player-dataset) on Kaggle.
